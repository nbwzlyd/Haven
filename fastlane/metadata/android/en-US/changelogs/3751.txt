v5.32.0 — selection, scrollback, prompt-aware tap

Selection
- Drag-select past the top or bottom of the viewport extends into
  scrollback (#94) — the viewport scrolls under your finger and the
  anchor stays on the line you originally pressed.
- Soft-wrap rejoin on copy now consults libvterm's authoritative
  softWrapped flag, so wrapped long commands paste back without
  spurious mid-line newlines, and hand-typed lines that happen to
  fill the viewport no longer get joined with the next line.
- Selection stays put during scroll — both anchors track their
  logical lines as the viewport moves underneath, matching desktop
  terminal behaviour. No more "I scrolled and lost my selection".

Scrollback
- New Settings → Terminal → Scrollback rows entry lets you bump
  per-tab history beyond the previous compile-time 1000 lines
  (#151). Presets 1 000 / 5 000 / 10 000 / 25 000; the dialog
  spells out the memory cost so you opt in deliberately.

Prompts
- New Settings → Terminal toggle "Tap to move cursor on supported
  prompts" — synthesises arrow-key dispatches when you tap inside
  the active prompt's input region. Requires OSC 133 prompt markers
  (starship, fish 3.6+, recent bash/zsh shell-integration setups).
  Default off.

Connections
- Session picker has a new "Open plain shell" row beneath "Create
  new session" — bypasses the profile's configured tmux / zellij /
  screen for one connection so a quick check doesn't disturb the
  long-running multiplexed session. Available from both the
  connect-time picker and the long-press-on-tab new-session picker
  (consolidated into a single dialog).
- Robot agent-active chip on the Connections header now turns
  green during the active window — distinct status colour rather
  than sharing the theme primary.

Migrations
- Schema-49 hash mismatch on v5.30 → v5.32 upgrade is fixed
  (rc2). The v5.31.0 ConnectionProfile column additions
  (autoReconnect / reconnectMaxAttempts / reconnectOnNetworkChange
  / tunnelOnly) are now picked up by a proper 49 → 50 migration.

Agent transport (MCP)
- 11 new tools for inspection and driving the terminal from the
  agent transport: structured snapshot reads, selection / scroll
  / tap / clipboard / preference round-trips. Settings writes and
  view-state mutations gated at ONCE_PER_SESSION; PTY input and
  clipboard writes stay EVERY_CALL.
- Headless agent shells opened by open_local_shell now register
  with the terminal session registry and survive the foreground
  UI's syncSessions sweep.
- list_connections DTO surfaces sessionManager and
  disableAltScreen so agents can pick a clean-bash test target
  without a probe connection.
