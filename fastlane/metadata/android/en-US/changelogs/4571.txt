Terminal keyboard-fix follow-up + WireGuard for the agent endpoint

Fixes a glitch from 5.59.4 where, with the soft keyboard up, the top of the terminal could paint over the status bar — the terminal is now clipped to its own area. (#206)

The MCP agent endpoint now binds directly on an active WireGuard tunnel when one is up (a stable address that survives network changes), and only falls back to the SSH reverse tunnel when no WireGuard tunnel is serving it.
